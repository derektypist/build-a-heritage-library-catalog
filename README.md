# Build a Heritage Library Catalog

Parse raw book data, search and group entries, validate data quality, and export results to different formats.

## Technologies

Uses JavaScript.

## Deployment

Copy `script.js` to the text editor of your choice.  If using terminal, use the command `node script.js` to run the code.
